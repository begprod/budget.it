## Features

- Automatic calendar generating
- Daily budget tracking
- Monthly budget tracking
- Last 6 month history
- Currencies setup
- Daily budget setup
- PWA available
- Import/export data

## How data is stored?

For this moment data is stored in local storage of your browser.
